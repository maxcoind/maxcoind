# Maxcoin


### Features:
 - Algorithm: Keccak (SHA-3) PoW
 - Signatures: Schnorr
 - Generation: 100 million MAX
 - Block time: 1 minute
 - Block Rewards: 8 coins per block
 - Halving every ~4 years
 - Difficulty Retarget: Every block KGW
